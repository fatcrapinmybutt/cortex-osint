## Anti-Patterns (15 Rules)

1. NEVER activate APEX skills without verifying GENESIS + DATAWEAVE are loaded (foundation dependency)
2. NEVER run APEX-WEBGPU on systems without WebGPU support — must check `navigator.gpu` first
3. NEVER persist raw evidence text in APEX-MEMORY localStorage — PII/security risk
4. NEVER let APEX-COGNITION agent loops run more than 10 iterations without human checkpoint
5. NEVER trust APEX-GRAPHML predictions above 0.85 confidence without manual verification
6. NEVER deploy APEX-VISION OCR results without lane classification validation
7. NEVER let APEX-DOCKET API calls exceed rate limits (MiCOURT: 100/hr, CourtListener: 1000/day)
8. NEVER include APEX-generated statistics in court filings without traceable DB query
9. NEVER allow APEX-AUTOMATON to auto-file without human QA review gate
10. NEVER mix APEX skill state across different case lanes (contamination risk)
11. NEVER delete skill files — archive to 11_ARCHIVES/ only (Rule 1 SACRED)
12. NEVER reduce APEX skill depth below 800 lines during updates
13. NEVER skip backup copies when creating or modifying APEX skills
14. NEVER bypass the SELFEVOLVE optimization loop — let the system learn
15. NEVER hardcode day counts — use `(today - date(2025,7,29)).days` dynamically

## Build Verification Protocol

After any skill creation or modification:

```python
import os, yaml, glob

SKILLS_DIR = r".agents\skills"
EXPECTED_COUNT = 33
MIN_LINES_APEX = 800
MIN_LINES_OTHER = 300

results = []
for skill_dir in sorted(glob.glob(os.path.join(SKILLS_DIR, "*", "SKILL.md"))):
    with open(skill_dir, "r", encoding="utf-8") as f:
        content = f.read()
    
    # Parse YAML frontmatter
    if content.startswith("---"):
        end = content.index("---", 3)
        meta = yaml.safe_load(content[3:end])
        desc_len = len(meta.get("description", ""))
    else:
        meta = {}
        desc_len = 0
    
    lines = len(content.splitlines())
    size_kb = len(content.encode("utf-8")) / 1024
    skill_name = os.path.basename(os.path.dirname(skill_dir))
    is_apex = "APEX" in skill_name
    min_lines = MIN_LINES_APEX if is_apex else MIN_LINES_OTHER
    
    results.append({
        "skill": skill_name,
        "lines": lines,
        "size_kb": round(size_kb, 1),
        "has_yaml": bool(meta),
        "desc_ok": desc_len <= 1024,
        "depth_ok": lines >= min_lines,
        "status": "✅" if (meta and desc_len <= 1024 and lines >= min_lines) else "❌"
    })

print(f"\n{'Skill':<45} {'Lines':>6} {'KB':>8} {'YAML':>5} {'Desc':>5} {'Depth':>6} {'Status':>7}")
print("-" * 90)
for r in results:
    print(f"{r['skill']:<45} {r['lines']:>6} {r['size_kb']:>8} {'✅' if r['has_yaml'] else '❌':>5} "
          f"{'✅' if r['desc_ok'] else '❌':>5} {'✅' if r['depth_ok'] else '❌':>6} {r['status']:>7}")

ok = sum(1 for r in results if r['status'] == '✅')
```
